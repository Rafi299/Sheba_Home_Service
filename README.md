# Sheba Home Service Management System

## Overview
Sheba Home Service Management System is a MERN Stack based web application that allows users to browse, book, and manage home services online. Administrators can manage users, services, bookings, and payments through a dedicated dashboard.

## Features

### User Features
- User Registration and Login
- JWT Authentication
- Browse Services
- Service Details View
- Add to Cart
- Book Services
- Payment Integration (SSLCommerz)
- Booking History
- Profile Management

### Admin Features
- Admin Dashboard
- Manage Users
- Manage Services
- Manage Bookings
- Upload Service Images
- Payment Monitoring

## Technologies Used

### Frontend
- Tailwind CSS
- React.js
- React Router
- Context API
- Axios
- Vite

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Multer

### Payment Gateway
- SSLCommerz
